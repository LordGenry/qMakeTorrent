## Description
qMakeTorrent is an advanced torrent creator with batch capability.

## Installation
### Linux
Requires a Qt4 development environment, C++/Boost 1.46+, and a recent version of libtorrent
    $ qmake
    $ make

Windows binaries can be found here.
